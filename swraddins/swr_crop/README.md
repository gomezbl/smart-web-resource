# Crop picly plugin

## Action name: _crop_

This Picly core plugin crop you to add charcoal effect to your images.

## Parameters

### w - integer

Width in pixels.

### h - integer

Height in pixels.

### x - integer

Horizontal offset in pixels.

### y - integer

Vertical offset in pixels.
